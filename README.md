# neis-api.ts
NEIS OPEN API를 쉽게 사용할 수 있는 API Wrapper 라이브러리 입니다.

## 설치 방법 (npm, yarn, pnpm)
```
npm install neis-api.ts
yarn add neis-api.ts
pnpm add neis-api.ts
```

## 엔드포인트 (12가지)
* AcaInsTiInfo
* ClassInfo
* ElsTimetable
* HisTimetable
* MealServiceDietInfo
* MisTimetable
* SchoolInfo
* SchoolMajorinfo
* SchoolSchedule
* SchulAflcoinfo
* SpsTimetable
* TiClrminfo
